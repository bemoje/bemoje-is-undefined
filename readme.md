## Installation

```sh
npm install @bemoje/is-undefined
npm install --save @bemoje/is-undefined
npm install --save-dev @bemoje/is-undefined
```

## Usage

```javascript
// import library
import isUndefined from '@bemoje/is-undefined'

isUndefined(void 0)
//=> true

isUndefined(undefined)
//=> true

isUndefined('undefined')
//=> false

isUndefined(null)
//=> false

```
